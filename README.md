## 📌 About This Project  

This project focuses on predicting the **resale price of used cars in Indian metro cities** using Machine Learning techniques.  

It involves:
- 📊 Exploratory Data Analysis (EDA)  
- 🧹 Data Cleaning & Feature Engineering  
- 🤖 Model Building (Decision Tree & Random Forest)  
- 📈 Price Prediction based on real-world features  

---

## 🎯 Objective  

- Build a **reliable ML model** for car price prediction  
- Understand **key factors affecting resale value**  
- Provide insights for **buyers & sellers**

## 📊 Key Insights  

- 🚗 **Premium brands** have higher resale prices  
- 💰 **Budget cars dominate in number**  
- ⛽ Petrol & Diesel cars are most common  
- 📉 Price decreases with **car age & mileage**  
- ⭐ Higher **quality score → higher price**  
- 🛡️ Cars with **warranty & 1st owner** tend to have better value
